# 請求書アプリの配布方法

## 1) まずは「1クリック起動」
- `起動.bat` をダブルクリックすると起動します。
- 初回のみ、必要な準備（依存関係インストール）を自動で行います。
- 2回目以降はそのままブラウザが開き、請求書アプリが使えます。

## 2) 配布先PCで必要なもの
- Windows
- Node.js (LTS)
  - 未インストールなら https://nodejs.org/ja から導入

## 3) 配布手順
1. このフォルダをZIP化して相手に渡す
2. 相手はZIPを展開
3. 展開したフォルダ内の `起動.bat` をダブルクリック

## 4) 完全にPC知識不要で配布する場合（推奨）
- Web公開してURLを配る方法が最も簡単です。
- 利用者は「URLをクリックするだけ」で使えます。
- もし希望があれば、次のステップで公開設定（Vercelなど）までこちらで実装・手順化します。
